# JuiceGoblin

전자담배 제품 정보 및 커뮤니티 웹사이트

## 프로젝트 소개

JuiceGoblin은 전자담배 제품 정보 및 최저가 사이트를 제공하고 커뮤니티를 지원하는 웹 플랫폼입니다.

관리자 기능을 통해 제품 관리가 가능하며, 사용자들은 커뮤니티를 통해 정보를 공유할 수 있습니다.

## 기술 스택

### 프론트엔드

- React 19
- Next.js 15
- Tailwind CSS
- TipTap 에디터
- React Icons
- Swiper

### 백엔드

- Next.js API Routes
- MySQL (with Sequelize ORM)
- Elasticsearch
- NextAuth.js (인증)
